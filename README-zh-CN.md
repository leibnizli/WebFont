# FontMagic
简体中文 | [English](./README.md)

FontMagic是一个macOS平台的字体处理客户端，它可以把字体文件中的指定字形导出为svg、pdf和png格式的文件，你可以在偏好设置中指定任意的颜色和尺寸。

![webFont 2](https://github.com/leibnizli/WebFont/assets/1193966/de9131c2-d5e4-4444-a3ee-2894c44c73ee)

## 快捷键

* 多选：`command+鼠标点击`
* 连选：`shift+鼠标点击`

## 其他
选中字形点击右键快速导出
从菜单栏批量导出选中的字形
